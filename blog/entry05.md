## Introduction 
After choosing my tool, ReactJS, I decided to research how to study it. I am really new to learning new code on my own, so it was a lengthy process. I first started by watching a YouTube video: [ReactJS for Beginners](https://www.youtube.com/watch?v=SqcY0GlETPk&t=2213s). But I then realized I wasn’t learning anything from the video, since this video is more meant for people who already know JavaScript, which I didn’t. I was so disappointed because I spent a long time studying using this video. I then tried to manually Google things in JavaScript that I might need, such as functions, but I didn’t truly understand each one. Eventually, I found [CodeAcademy](https://www.codecademy.com/). Code Academy had a JavaScript course, so I started one of its lessons, but then my friend told me there was a whole course just for ReactJS. So, I continued the course where I learned how to render and make variables in ReactJS.  For example, when you write a variable, you can write `const insertNameOfVariable =` and then put your code right afterwards. To render the code, you can write `root.render(insertNameOfVariable)`. If you wanna write HTML code in a function, you put `return()` and write your code in parentheses, if you want to turn it back into JavaScript code, you use the curly brackets: `{}`. I still don’t feel like I fully mastered ReactJS, but as I am working on my freedom project, I will probably get the hang of it much more!
